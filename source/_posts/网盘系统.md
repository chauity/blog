# 项目介绍

## 项目描述

实现一个类似百度网盘的网络存储系统。功能包括文件同步、备份和共享等网络平台云服务，可以随时把照片、音乐、视频、文档等轻松保存到网盘。用户可以多终端下载、管理、分享文件。

在loadConfig方法上按住**Alt+Enter**，在生成其对应的方法实现

1.加载配置文件

2.只读方式打开文件 

3.循环获取文件内容

5.分割获得解析的字符串 （以 空格 为分隔符）

完整方法实现如下

```cpp
void TcpClient::loadConfig()
{
    //1.加载配置文件
    QFile file(":/client.config");
    //2.只读方式打开文件 
    if(file.open(QIODevice::ReadOnly))
    {
        //3.循环获取文件内容 
        QByteArray baData = file.readAll();
        QString strData = baData.toStdString().c_str();
        file.close();
        // 4.将\r\n 替换为 空格字符串
        strData.replace("\r\n"," ");
        // 5.以空格进行分割
        QStringList strList = strData.split(" ");
        m_strIP = strList.at(0);
        m_usPort = strList.at(1).toUShort();
    }
    else
    {
        QMessageBox::critical(this, "open config", "open config failed");
    }
}
```

# 通讯协议设计

## 弹性结构体

此小节为示例，可不做

**PDU**是**protocal data unity**的简写，是协议数据单元的意思

``` cpp
typedef struct PDU
{
    int a;
    int b;
    int c;
    int d[];
}PDU;
```

使用数组实现 结构体的大小 弹性申请空间

实例代码：main.c

```cpp
#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>
#include <string.h>

typedef struct PDU
{
    int a;
    int b;
    int c;
    int d[];
}PDU;

int main(int argc, char* argv[])
{
    printf("%ld\n", sizeof(PDU));
    // 动态申请空间
    PDU* pdu = (PDU*) malloc((sizeof(PDU)) + (100*sizeof(int)));
    // 赋值
    pdu->a = 90;
    pdu->b = 80;
    pdu->c = 70;
    memcpy(pdu->d, "You jump I jump", 16);
    printf("a=%d, b=%d, c=%d, d=%s", pdu->a, pdu->b, pdu->c, (char*) pdu->d);
    // 释放空间
    free(pdu);
    pdu = NULL;
    return 0;
}
```

- 弹性结构体原理 => 结构体最后一个成员为 int caData[];

## 通讯协议设计

- 协议设计

1.总的消息大小

2.消息类型

3.实际消息大小

4.实际消息

| 总的消息大小 | 消息类型  | 实际数据   | 实际消息大小 | 实际消息 |
| ------------ | --------- | ---------- | ------------ | -------- |
| uiPDULEN     | uiMsgType | caData[64] | uiMsgLen     | caMsg[]  |

在两个项目（**TcpClient、TcpServer**）中新建协议类

protocol.h

```cpp
#ifndef PROTOCAL_H
#define PROTOCAL_H
#include <stdlib.h>
#include <unistd.h>
#include <string.h>

typedef unsigned int uint;

struct PDU
{
    uint uiPDULEN;      // 总的协议数据单元大小
    uint uiMsgType;     // 消息类型
    char caData[64];    //
    uint uiMsgLen;      // 实际消息长度
    int caMsg[];        // 实际消息
};

// 构造变长的PDU
PDU* mkPDU(uint uiMsgLen);

#endif // PROTOCAL_H
```

protocol.cpp

```cpp
#include "protocol.h"

PDU* mkPDU(uint uiMsgLen)
{
    uint uiPDULen = sizeof(PDU) + uiMsgLen;
    PDU* pdu= (PDU*)malloc(uiPDULen);
    if(NULL == pdu)
    {
        exit(EXIT_FAILURE);
    }
    memset(pdu, 0, uiPDULen);
    pdu->uiPDULEN = uiPDULen;
    pdu->uiMsgLen = uiMsgLen;
    return pdu;
}
```

#include "mytcpsocket.h"
#include <QDebug>

MyTcpSocket::MyTcpSocket()
{
    connect(this, SIGNAL(readyRead()),
            this, SLOT(recvMsg()));
}

void MyTcpSocket::recvMsg()
{
    qDebug() << this->bytesAvailable();
    uint uiPDULen = 0;
    this->read((char*)&uiPDULen, sizeof(uint));
    // 实际消息长度
    uint uiMsgLen = uiPDULen - sizeof(PDU);
    PDU* pdu = mkPDU(uiMsgLen);
    this->read((char*)pdu+sizeof(uint), uiPDULen - sizeof(uint));
    qDebug() << pdu->uiMsgType << (char*)(pdu->caMsg);
}
